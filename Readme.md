## Hey there 👋

I'm **Sifat Ibna Amin** — a **Senior Software Engineer** at [Intelligent Machines](https://intelligentmachines.tech) and a self-proclaimed **polyglot programmer** who genuinely enjoys switching between Python, TypeScript, Java, and C++ depending on what the problem demands.

Over the past **4+ years**, I've been building things that sit at the intersection of **AI, blockchain, and cloud-native architecture** — not because they're buzzwords, but because these are the tools that let me solve the kinds of problems I find most interesting: how do you make systems that are smart, trustworthy, and can handle real scale?

I've shipped platforms that serve **thousands of users across Asia**, built blockchain payment gateways processing enterprise transactions across Japan, China, and Korea, and designed AI agents that monitor and optimize themselves. I graduated from **BUET** (Bangladesh University of Engineering and Technology) with a CS degree, and I've been obsessed with building things ever since.

> *"I think in systems, write in multiple languages, and believe the best code is the code that scales itself."*

## 🔨 What I Build

I don't just write code — I architect systems. Here's what I'm currently deep into:

**🤖 AI & Co-Learning Platforms** — Building intelligent systems that go beyond simple automation. My current focus is on **Co-Learning platforms** that use AI to reshape how people learn together — think real-time speech analysis, adaptive feedback loops, and collaborative environments powered by AI agents. I've also built computer vision pipelines with **YOLOv5** that hit 95% accuracy in retail analytics.

**⛓️ Blockchain & Decentralized Identity** — I architect **blockchain-based payment gateways** and **DID (Decentralized Identity) certification systems**. The payment work serves 4,500+ companies with multi-language support across East Asia. The DID work is about giving institutions a secure, decentralized way to manage credentials — no single point of failure, no single point of trust.

**☁️ Cloud Architecture & DevOps** — I run **MicroK8s clusters with 99.9% uptime**, design CI/CD pipelines with Jenkins and ArgoCD that cut deployment time by 80%, and build multi-cloud solutions across AWS and Azure. I believe infrastructure should be invisible — it should just work, scale, and recover.

**📊 Data Engineering** — I've built enterprise pipelines processing **1M+ daily transactions** with Apache Airflow, optimized PostgreSQL for 70% faster retrieval, and implemented Redis caching that delivers sub-second response times on concurrent financial systems.
